## ΏθΥΝ Overview

CinemaIQ is a full-stack machine learning application that predicts **box office revenue for South Asian films** έΑΦ Bollywood, Tollywood, Kollywood, and beyond. It models how budget, cast, director, composer, genre, and release timing jointly drive revenue, and surfaces actionable production strategy insights through an interactive Streamlit dashboard.

---

## έερ Features

| Tab | What it does |
|-----|--------------|
| **ΏθΟψ Prediction Engine** | Input your film's parameters and get pessimistic / base / optimistic revenue forecasts with ROI breakdown |
| **έγΦΎ╕Π Strategy Battle** | Head-to-head comparison of two production configurations έΑΦ revenue, ROI gauges, and a winner verdict |
| **ΏθΥΛ Market Insights** | Model-derived patterns: optimal release months, genre revenue rankings, top actor & director power indices |

**Throughout all tabs**, an AI Analyst (Groq LLaMA 3.3 70B) delivers concise, data-grounded commentary explaining the *why* behind every number.

## Ώθπι Model

Five regression models were trained and evaluated on a held-out test set. The best-performing model is auto-selected and saved.

| Model | R┬▓ | MAE |
|-------|-----|-----|
| **XGBoost** έφΡ | *0.XX* | *$XX.XM* |
| Gradient Boosting | έΑΦ | έΑΦ |
| Random Forest | έΑΦ | έΑΦ |
| Decision Tree | έΑΦ | έΑΦ |
| Linear Regression | έΑΦ | έΑΦ |

> Fill in your actual metrics after running `train.ipynb`.

**Target variable:** `log1p(revenue)` έΗΤ inverted back to dollars at inference time.

**Input features used:**

- `log_budget` έΑΦ production budget (log-scaled)
- `runtime` έΑΦ film length in minutes
- `vote_average` έΑΦ IMDb-style audience rating
- `vote_count` έΑΦ proxy for audience reach
- `release_month` έΑΦ 1έΑΥ12
- `star_power` έΑΦ mean actor power index across cast
- `director_power` έΑΦ director historical performance index
- `composer_power` έΑΦ composer historical performance index
- `Genre_*` έΑΦ multi-hot encoded genre columns

## έγβΎ╕Π Setup

### 1. Clone & install

```bash
git clone https://github.com/your-username/cinemaiq.git
cd cinemaiq
pip install -r requirements.txt
```

### 2. Add your Groq API key

Create a `.env` file in the project root:

```env
GROQ_API_KEY=your_groq_key_here
```

Get a free key at [console.groq.com](https://console.groq.com).  
The app works without it έΑΦ AI insight cards are silently skipped if the key is missing.

### 3. Run the pipeline

```bash
# Open and run all cells in order:
jupyter notebook cleaningdata.ipynb
jupyter notebook preprocess.ipynb
jupyter notebook train.ipynb
```

This generates `movie_revenue_model.pkl` and `model_artifacts.pkl`.

### 4. Launch the app

```bash
streamlit run app.py
```

## ΏθγΑ Deploy to Streamlit Cloud

1. Push the repo to GitHub (include `.pkl` files or generate them on deploy)
2. Go to [share.streamlit.io](https://share.streamlit.io) έΗΤ **New app**
3. Set the main file to `app.py`
4. Add `GROQ_API_KEY` under **Advanced settings έΗΤ Secrets**

```toml

GROQ_API_KEY = "your_groq_key_here"
```

## ΏθΥΙ How Scenario Forecasting Works

Given base inputs, the app generates three scenarios:

| Scenario | Logic |
|----------|-------|
| **Pessimistic** | Rating έΙΤ0.8, vote count ├Ω0.55, output ├Ω0.78 |
| **Base** | Inputs as-is |
| **Optimistic** | Rating +0.7, vote count ├Ω1.6, output ├Ω1.22 |

ROI is calculated as `(predicted_revenue έΙΤ budget) / budget ├Ω 100`.
